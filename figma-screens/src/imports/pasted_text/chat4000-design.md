Design a native iOS and macOS chat app called "chat4000" with a dark, minimalist aesthetic inspired by Notion and iMessage. Dark mode only. Clean sans-serif typography (SF Pro), generous whitespace, subtle   
  animations.                                                                                                                                                                                                        
                                                                                                                                                                                                                     
  Screen 1 — Setup Screen (first launch / no saved connection)                                                                                                                                                       
                                                                                                                                                                                                                     
  Full-screen centered card on a near-black background (#0F0F0F). The card contains:                                                                                                                                 
  - App icon at top: a minimal line-art claw/link symbol in accent blue (#4A9EFF)                                                                                                                                    
  - "chat4000" title in large bold white text                                                                                                                                                                     
  - Three input fields with subtle dark borders (#1E1E1E background, #2A2A2A border), rounded corners:                                                                                                               
    - "Server" (placeholder: agent.example.com)                                                                                                                                                                      
    - "Port" (placeholder: 18789)                                                                                                                                                                                    
    - "Token" (placeholder: oc_tok_..., secure/masked field)                                                                                                                                                         
  - A prominent "Connect" button: solid accent blue, full-width, rounded, white text                                                                                                                                 
  - Below the button: subtle status text area for errors ("Connection failed — check your server address") in red (#FF4A4A)                                                                                          
                                                                                                                                                                                                                     
  On macOS: this card is centered in a 400x500 window. On iOS: full-screen with the card vertically centered.                                                                                                        
                                                                                                                                                                                                                     
  Screen 2 — Chat Screen (main screen after connection)                                                                                                                                                              
                                                                                                                                                                                                                     
  Full-screen chat interface on near-black background.                                                                                                                                                               
                                                                                                                                                                                                                     
  Top nav bar:                                                                                                                                                                                                       
  - Left: "chat4000" title in medium white text                                                                                                                                                                   
  - Right: a small colored dot indicating connection status (green = connected, yellow = reconnecting, red = disconnected), and a gear icon for settings                                                             
                                                                                                                                                                                                                     
  Message area (scrollable, fills remaining space):                                                                                                                                                                  
  - User messages: right-aligned, accent blue (#4A9EFF) bubble, white text, rounded corners (larger radius on the user's side). Subtle timestamp below in grey (#666).                                               
  - Agent messages: left-aligned, dark grey (#1A1A1A) bubble, light grey text (#E0E0E0), rounded corners. Small "AI" or claw icon to the left of the first message in a sequence. Subtle timestamp below.            
  - Messages have comfortable vertical spacing (12px between messages, 24px between message groups from different senders).                                                                                          
  - Auto-scrolls to bottom on new messages.                                                                                                                                                                          
                                                                                                                                                                                                                     
  Bottom input area:                                                                                                                                                                                                 
  - Dark input bar fixed to bottom (#141414 background)                                                                                                                                                              
  - Text input field: dark (#1E1E1E), rounded, placeholder "Message..." in grey                                                                                                                                      
  - Send button: accent blue circle with white arrow-up icon, appears only when text is entered                                                                                                                      
  - On iOS: proper keyboard avoidance with smooth animation                                                                                                                                                          
  - On macOS: Cmd+Return sends, Return adds newline                                                                                                                                                                  
                                                                                                                                                                                                                     
  Screen 3 — Settings Sheet                                                                                                                                                                                          
                                                                                                                                                                                                                     
  Bottom sheet on iOS, popover on macOS. Dark background (#141414).                                                                                                                                                  
  - Server info displayed (host, port) with an "Edit" button that returns to setup screen                                                                                                                            
  - "Disconnect" button in red                                                                                                                                                                                       
  - "Clear Chat History" button with a confirmation dialog                                                                                                                                                           
  - App version at bottom in small grey text                                                                                                                                                                         
                                                                                                                                                                                                                     
  General design notes:                                                                                                                                                                                              
  - All transitions are smooth, 0.2s ease-in-out                                                                                                                                                                     
  - No sharp borders anywhere — everything softly blends with the dark background                                                                                                                                    
  - Message bubbles have very subtle shadow/glow                                                                                                                                                                     
  - The overall feeling should be: calm, focused, premium, like a private terminal to your AI
